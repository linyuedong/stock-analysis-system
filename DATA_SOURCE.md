# 数据源说明

## 当前数据源
本系统使用 **Yahoo Finance** 作为股票数据源，通过 Python 的 `yfinance` 库获取数据。

### 优势
- ✅ **免费使用**：无需 API Key 或付费订阅
- ✅ **全球覆盖**：支持美股、港股、A股及其他主要市场
- ✅ **数据丰富**：包含历史价格、成交量等基础数据

### 限制
- ⚠️ **延迟数据**：免费版本有 15-20 分钟延迟（非实时）
- ⚠️ **稳定性**：偶尔会出现访问限制或数据缺失
- ⚠️ **速率限制**：频繁请求可能被限流

## 支持的市场

### 🇺🇸 美股（美国）
- **代码格式**：直接使用股票代码
- **示例**：
  - `AAPL` - Apple Inc.
  - `NVDA` - NVIDIA Corporation
  - `TSLA` - Tesla, Inc.

### 🇨🇳 A股（中国大陆）
- **上海交易所**：股票代码 + `.SS`
  - `600519.SS` - 贵州茅台
  - `600036.SS` - 招商银行
  - `601318.SS` - 中国平安

- **深圳交易所**：股票代码 + `.SZ`
  - `000001.SZ` - 平安银行
  - `000858.SZ` - 五粮液
  - `002594.SZ` - 比亚迪

### 🇭🇰 港股（香港）
- **代码格式**：4位数字代码 + `.HK`
- **示例**：
  - `0700.HK` - 腾讯控股
  - `9988.HK` - 阿里巴巴-SW
  - `1024.HK` - 快手-W
  - `2318.HK` - 中国平安（H股）

## 替代数据源

如果需要更专业的数据服务，可考虑：

### 1. **Alpha Vantage**
- 免费层级：500 请求/天
- 支持实时数据（付费）
- 官网：https://www.alphavantage.co/

### 2. **Tushare（推荐用于A股）**
- 专注于中国市场
- 提供更丰富的A股数据
- 官网：https://tushare.pro/

### 3. **AKShare**
- 开源免费
- 专注于中文金融数据
- GitHub：https://github.com/akfamily/akshare

## 如何切换数据源

如果需要切换到其他数据源，只需修改 `backend/services/stock_service.py` 文件中的数据获取逻辑即可。
